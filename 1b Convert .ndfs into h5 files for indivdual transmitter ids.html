Converting ndfs¶

Here we will convert ndfs into .h5 files. Ndf files can contain more than one transmitter, so we will split the ndf files into sub folders of h5 files in which each h5 contains just one transmitter.

This will also make use of the NdfFile class

In [4]:

root_h5_dir     = '/home/jonathan/Desktop/extraction_test'
ndf_folderpath  = '/home/jonathan/Desktop/test_ndfs'

ndf_filepaths = [os.path.join(ndf_folderpath,f) for f in os.listdir(ndf_folderpath)]

Loop through first 5 files and print transmitter ids and sampling frequency.

Note, to calculate sampling frequency pyecog assumes that each file contains 1 hour of uninterupted samples.

As a first step, work out the transmitter ids contained in our folder of ndfs. We can then create a set of folders in the ‘root_h5_dir’ to contain h5 files for each transmitter.

In [11]:

# first obtain set of transmitter ids
transmitter_ids = set()
for filepath in ndf_filepaths:
    ndf_file = pg.NdfFile(filepath)
    ndf_file.get_valid_tids_and_fs()

    transmitter_ids = transmitter_ids.union(ndf_file.tid_set)

    handler.convert_ndf_directory_to_h5(ndf_dir  = ndf_folderpath,
